物理设备和队列家族

概要

在创建好了Instance之后，我们需要选择一个合适的物理设备（显卡）来执行实际的操作。实际上我们可以选择多张显卡同时工作，不过简单起见本文只选择一张显卡进行操作。选择合适的显卡主要分为两步：

  • 决定应用程序需要显卡具有的功能和属性。
  • 从所有显卡中选择合适的那个作为物理设备。

本文对应Vulkan Tutorial的Physical devices and queue families章节。

显卡要求

显卡在Vulkan中的对应句柄对象为VkPhysicalDevice，我们可以创建一个bool isDeviceSuitable(VkPhysicalDevice device)函数来决定一张显卡是否满足我们的需求。目前我们只需要显卡支持图形队列家族（Graphics queue families），也就是能够接受图形有关命令的队列家族，于是对于一个给定的VkPhysicalDevice，我们可以创建一个QueueFamilyIndices findQueueFamilies(VkPhysicalDevice device)函数来检查它是否支持该队列家族。其中，我们定义结构体QueueFamilyIndices 如下：
