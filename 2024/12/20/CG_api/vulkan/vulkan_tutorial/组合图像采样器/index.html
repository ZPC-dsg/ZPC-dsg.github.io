组合图像采样器

概要

现在我们已经准备好了采样纹理所需的一切对象，包括纹理本身、纹理视图和纹理采样器。本节我们会将这些对象通过组合图像采样器这一特殊的描述符连结起来，最终完成着色器纹理采样的工作。主要过程可以分为三步：

  • 创建组合图像采样器描述符
  • 调整顶点结构以传递纹理坐标
  • 在着色器中使用纹理采样

本节对应Vulkan Tutorial的Combined image sampler章节。
